# exp, dexp



返回 e 的 `x` 次幂，也被称为自然指数函数。

## 语法

```Haskell
EXP(x);
```

## 参数说明

`x`: 支持的数据类型为 DOUBLE。

## 返回值说明

返回值的数据类型为 DOUBLE。

## 示例

```Plain Text
mysql> select exp();
+--------------------+
| exp()          |
+--------------------+
|  |
+--------------------+
1 row in set ( sec)
```
