# degrees



将参数 `x` 转成角度，`x` 是弧度。

## 语法

```Haskell
DEGREES(x);
```

## 参数说明

`x`: 支持的数据类型为 DOUBLE。

## 返回值说明

返回值的数据类型为 DOUBLE。

## 示例

```Plain Text
mysql> select degrees();
+--------------------------+
| degrees() |
+--------------------------+
|         |
+--------------------------+
1 row in set ( sec)
```
