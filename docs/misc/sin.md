# sin



计算参数 `x` 的正弦，`x` 为弧度值。

## 语法

```Haskell
SIN(x);
```

## 参数说明

`x`: 支持的数据类型为 DOUBLE。

## 返回值说明

返回值的数据类型为 DOUBLE。

## 示例

```Plain Text
mysql> select sin();
+-----------------------+
| sin()             |
+-----------------------+
|  |
+-----------------------+
1 row in set ( sec)
```
